STEP 3 — Signal computation functions
Create artifacts/api-server/src/lib/variance/compute-variance.ts

Implement exactly as specced:

computeFatigueScore(data): { score, label, warnings }
Factors: isBackToBack (+35), isThreeInFour (+20),
travelMiles (up to +15), timezoneShift (up to +10),
prevGameMinutes (up to +10), prevGameWasOT (+8),
isEarlyGame (+5), well rested (-10).
Cap at 0-100.

computeBlowoutRisk(spread, total, sport):
{ probability, warning, evAdjustment }
NBA thresholds: moderate 9, heavy 14, extreme 18.
NFL: moderate 10, heavy 14, extreme 17.
MLB: moderate 2.5, heavy 3.5, extreme 5.
NHL: moderate 1.5, heavy 2, extreme 2.5.

computeUsageDelta(playerId, statType):
{ score, usageDelta, minutesTrend, label }
Compare last 5 games vs season average.
Score 0-100 where 50 is neutral.

computeEVModifier(signals): number
Fatigue >= 60: -0.06
Fatigue >= 40: -0.03
Fatigue <= -5: +0.02
Blowout adjustment: direct from computeBlowoutRisk
Usage delta > 15%: +0.06
Usage delta > 8%: +0.03
Usage delta < -15%: -0.05
Usage delta < -8%: -0.02
Aggressive mode: multiply by 2
Hard cap: Math.max(-0.15, Math.min(0.15, modifier))

STEP 4 — Master variance compute function
Create artifacts/api-server/src/lib/variance/index.ts

computeVarianceForLine(ppLineId): void
1. Fetch pp_line + player
2. Get fatigue from fatigue_data table
3. Get spread + total from game_environment
4. Compute blowout risk
5. Compute usage delta from player_game_logs
6. Compute matchup score from matchup_history
7. Compute EV modifier from validated signals only
8. Build whyItMoves human-readable summary
9. Upsert to variance_scores

computeAllVarianceScores(): number
Loop all active pp_lines and compute variance.
Return count of computed scores.
